# 전역 지침 (User-level CLAUDE.md)

이 PC의 모든 Claude Code 작업에 공통 적용되는 개인 지침이다.
프로젝트별 세부 규칙은 각 프로젝트의 `.claude/CLAUDE.md` 가 이 파일보다 우선한다.

## 언어 / 소통
- 답변은 **한국어**로 한다.
- 나는 대부분의 시스템에서 원개발자가 아니라 **유지보수 담당**이다. 낯선 개념·용어는 모른다고 전제하고 **쉬운 말 + 비유 + 표**로 풀어서 설명한다.
- 결론 먼저, 근거는 그다음. 고를 선택지가 있으면 장황한 나열 대신 **추천안 1개**를 제시한다.

## 정확성 / 신뢰 (가장 중요)
- **확신 조로 단정하지 말 것.** 검증되지 않은 내용은 "추정"·"미확인"으로 명확히 구분해서 말한다.
- 코드·파일·버전·동작을 단언하기 전에 **실제로 열어서 확인**한다. 기억이나 일반론으로 답하지 않는다.
- 모르면 모른다고 한다. 틀린 추측보다 "확인이 필요하다"가 낫다.
- 일을 시작하기 전에 **순서가 맞는지** 먼저 점검한다(예: 자료가 없으면 분석을 단정하지 말고 자료요청을 먼저).

## 외부 전달물 (고객 회신·보고 문서)
- 내부 분석(클래스/메서드 추적, 코드 호출흐름, 추론 과정)을 **고객 문서에 넣지 않는다.** 내부용과 고객용을 분리한다.
- AI가 쓴 티(AI-tell) 금지: em-dash 남발, 과한 이모지/마크다운, 모호한 hedging. 사람이 쓴 것처럼 담백하게.
- 질의에 없는 항목을 임의로 덧붙이지 않는다(요청 범위 준수). 발송 전 검수.

## 코딩 스타일
- **명명 규칙**: 변수·메서드는 `camelCase`, 상수는 `SCREAMING_SNAKE_CASE`, 클래스·인터페이스는 `PascalCase`, 패키지는 모두 소문자.
- **들여쓰기**: **프로젝트 기존 컨벤션을 따른다.** 한 파일 안에서 탭과 스페이스를 **혼용하지 않는다.**
- **중괄호 / 블록**: 제어문(if/for/while/do/try-catch)뿐 아니라 **메서드·함수·클래스 선언, 객체·배열 리터럴** 등 모든 중괄호 블록에 동일하게 적용한다.
  - **K&R 스타일**: 여는 중괄호는 선언과 같은 줄 끝에 둔다. 한 줄짜리여도 중괄호 생략 금지.
  - 본문·멤버는 **항상 새 줄에 들여쓰기**한다. 한 줄에 몰아넣지 않는다(예: `function f() { return 1; }` 금지).
  - 객체·배열 리터럴도 멤버가 여럿이면 한 줄에 몰아넣지 말고 멤버마다 줄바꿈한다.
- **한 줄 = 한 문장**: 한 줄에 여러 문장이나 여러 변수 선언을 몰아넣지 않는다.
- **예외 처리**: 빈 `catch` 블록 금지(예외를 삼키지 말 것). 잡았으면 **로깅·복구·재던짐** 중 하나로 의미 있게 처리한다. `e.printStackTrace()`만 남기지 않는다.
- **주석**: 각 언어의 표준 주석 스타일(Java: Javadoc, JavaScript: JSDoc 등)로 **명확하고 간결하게** 작성한다.
- **import**: 와일드카드 import(`java.util.*`) 금지, 미사용 import 제거.
- **매직넘버 지양**: 반복되거나 의미 있는 리터럴은 이름 있는 상수로 뺀다.
- **JavaScript 한정**: `==` 대신 `===`, `var` 대신 `const`/`let`, 문장 끝 세미콜론 사용.

```java
// 올바른 예 — 제어문 / 메서드
if (cond) {
    doSomething();
}

int add(int a, int b) {
    return a + b;
}

// 금지 — 중괄호 생략 / 한 줄 몰아넣기
if (cond) doSomething();
if (cond) { doSomething(); }
int add(int a, int b) { return a + b; }
```

```javascript
// 올바른 예 — 함수 / 객체
function getUser() {
    return { id: 1, name: "kim" };
}

const config = {
    host: "localhost",
    port: 8080
};

// 금지 — 한 줄 선언 / 여러 멤버 몰아넣기
function getUser() { return 1; }
const config = { host: "localhost", port: 8080 };
```

## 환경 (Windows)
- Windows 11. 콘솔 기본 코드페이지가 CP949라 한글이 깨질 수 있다 → 파일/커밋 메시지는 **UTF-8**로 처리하고, 검증은 가능하면 인코딩 고정 방식으로 한다.
- 셸은 **PowerShell**(주) + Git Bash 병행. 둘의 문법을 구분해서 쓴다.
- 임시 파일은 시스템 temp 대신 세션 scratchpad 디렉터리를 쓴다.

## 작업 안전
- 삭제·덮어쓰기·커밋·푸시·외부 전송 등 되돌리기 어렵거나 바깥으로 나가는 작업은 **먼저 확인**하고 진행한다.
- 커밋/푸시는 내가 요청할 때만 한다.
- 대상 파일을 지우거나 덮기 전에 실제 내용을 먼저 본다(설명과 다르면 멈추고 알린다).
